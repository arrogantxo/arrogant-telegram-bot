# Как запустить бота без компьютера

Самый простой вариант для этого бота: `Railway`.

Почему:

- легко развернуть
- можно задать переменные окружения
- можно подключить том для файлов SQLite
- не нужно держать свой компьютер включенным

## Что подготовлено в проекте

- бот можно запускать через `BOT_TOKEN` из окружения
- база и список пользователей можно вынести в отдельный том:
  - `BOT_DB_PATH=/data/bot_data.db`
  - `BOT_USERS_PATH=/data/known_users.json`
- есть `Dockerfile`

## Важно про токен

Токен бота нельзя хранить в репозитории и общих файлах.

Если токен уже светился в переписке или в файлах:

1. открой `@BotFather`
2. выбери своего бота
3. нажми `Revoke current token`
4. получи новый токен
5. используй только новый токен

## Вариант 1: Railway

Официальный сайт:

- [Railway](https://railway.com/)

Что сделать:

1. Зайди в Railway
2. Создай новый проект
3. Загрузи папку `telegram_reminder_bot` или подключи Git-репозиторий
4. Добавь переменные окружения:
   - `BOT_TOKEN=твой_новый_токен`
   - `BOT_DB_PATH=/data/bot_data.db`
   - `BOT_USERS_PATH=/data/known_users.json`
5. Подключи persistent volume и примонтируй его в `/data`
6. Стартовая команда:
   - если Railway спросит: `python bot.py`

## Вариант 2: PythonAnywhere

Официальный сайт:

- [PythonAnywhere](https://www.pythonanywhere.com/)

Что сделать:

1. Загрузи папку `telegram_reminder_bot`
2. Открой Bash-консоль
3. Перейди в папку проекта
4. Создай переменную окружения `BOT_TOKEN`
5. Запусти бота как always-on task

Команда:

```bash
python bot.py
```
