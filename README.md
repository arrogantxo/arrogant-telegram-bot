# Telegram Reminder Bot

Общий Telegram-бот для команды. Он напоминает о рабочих задачах и дедлайнах.

## Для чего нужен

- напоминания о съемках
- напоминания о сценариях
- напоминания о монтаже
- напоминания о публикациях
- напоминания о созвонах
- напоминания о согласованиях и выездах

## Что умеет

- отправлять личные напоминания сотрудникам
- отправлять напоминания в общий чат
- хранить задачи в SQLite
- показывать список активных задач
- показывать задачи на сегодня
- отмечать задачи выполненными
- удалять задачи
- показывать список сотрудников
- показывать список категорий

## Команды

- `/start` - приветствие
- `/help` - список команд
- `/myid` - показать chat id
- `/team` - список сотрудников из конфига
- `/categories` - список категорий задач
- `/users` - пользователи, которые уже писали боту
- `/add YYYY-MM-DD HH:MM employee_key category text`
- `/broadcast YYYY-MM-DD HH:MM category text`
- `/list`
- `/today`
- `/done TASK_ID`
- `/delete TASK_ID`

## Пример

```text
/add 2026-04-10 10:00 videograf съемка Съемка reels для O'zbegim Mall
/add 2026-04-10 14:00 scriptwriter сценарий Написать сценарий для discount reels
/broadcast 2026-04-11 09:00 созвон Общая планерка по контенту
```

## Настройка

1. Создай Telegram-бота через `@BotFather`
2. Заполни `config.json`
3. Впиши токен бота
4. Впиши `chat_id` админов
5. Впиши `chat_id` сотрудников
6. При необходимости впиши `chat_id` общего чата

## Как узнать chat id

Каждый сотрудник пишет боту:

```text
/start
/myid
```

После этого бот покажет его `chat_id`.

Если не хочется собирать `chat_id` вручную:

- пусть сотрудники просто напишут боту `/start`
- бот автоматически сохранит их в файл `known_users.json`
- потом можно посмотреть их через команду `/users`
