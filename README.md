# Telegram Reminder Bot

Удобный Telegram-бот для команды. Он помогает ставить задачи обычными сообщениями, показывает задачи сотрудникам и теперь позволяет добавлять новые роли прямо из бота.

## Быстрый старт

Пиши боту обычным языком:

```text
Завтра съемка в 12, монтажер и видеограф
Сегодня в 18:00 публикация, publisher
Завтра в 10 сценарий для сценариста
Напомни всей команде о съемке завтра в 9
Какие задачи на сегодня?
Мои задачи
```

## Что умеет бот

- создавать задачи из обычных сообщений
- отправлять личные напоминания сотрудникам
- отправлять общее напоминание в командный чат
- показывать задачи на сегодня
- показывать свои задачи сотруднику
- принимать голосовые сообщения
- распознавать голос через OpenAI
- добавлять и удалять роли через Telegram

## Основные команды

- `/start`
- `/help`
- `/myid`
- `/team`
- `/roles`
- `/categories`
- `/users`
- `/list`
- `/today`
- `/done TASK_ID`
- `/delete TASK_ID`
- `/addrole role_key chat_id Имя | алиас1,алиас2`
- `/delrole role_key`

## Самый удобный способ ставить задачи

Вместо длинных команд просто пиши:

```text
Завтра съемка в 12, Абдулло и Мухаммадаюб
Сегодня в 18 публикация, publisher
Завтра в 10 сценарий для сценариста
```

## Как добавить нового человека через бота

### Шаг 1

Пусть человек напишет боту:

```text
/start
/myid
```

### Шаг 2

Возьми его `chat_id`.

### Шаг 3

Добавь роль командой:

```text
/addrole assistant3 123456789 Саид | саид,ассистент,said
```

Что значит:

- `assistant3` — ключ роли
- `123456789` — chat id человека
- `Саид` — имя для отображения
- `саид,ассистент,said` — слова, по которым бот будет его узнавать

### Шаг 4

Проверь список ролей:

```text
/roles
```

### Шаг 5

Теперь можно писать:

```text
Завтра съемка в 12, Саид
Сегодня в 18 публикация, assistant3
```

## Как удалить роль

```text
/delrole assistant3
```

Через эту команду удаляются только роли, которые были добавлены через `/addrole`.

## Что могут сотрудники

Сотрудник может:

- написать `/start`
- узнать свой `chat_id` через `/myid`
- спросить `Какие задачи на сегодня?`
- спросить `Мои задачи`

Сотрудник не создаёт новые задачи, если он не админ.

## Голосовые сообщения

Примеры голосовых:

- "Завтра съемка в 12, монтажер и видеограф"
- "Какие задачи на сегодня?"
- "Напомни всей команде о съемке завтра в 9"

Важно:

- для голосовых нужен `OPENAI_API_KEY`
- если у OpenAI нет квоты или billing, голос не распознается

## Railway Variables

В Railway Variables должны быть:

```text
BOT_TOKEN=...
BOT_DB_PATH=/data/bot_data.db
BOT_USERS_PATH=/data/known_users.json
BOT_CUSTOM_ROLES_PATH=/data/custom_roles.json
OPENAI_API_KEY=sk-...
```

## Volume

У Railway volume должен быть подключен к:

```text
/data
```
